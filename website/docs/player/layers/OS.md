# OS
Every node will always have an OS layer as layer 0. The OS manages things like the general operation of the node and the connections to other nodes.

The OS does not contain information and does not protect the system. Therefore, hacking it is never useful.

```
⇋ view 
Node service layers: 
0 OS 

⇋ hack 0 
Hacking OS reveals nothing new.
```
