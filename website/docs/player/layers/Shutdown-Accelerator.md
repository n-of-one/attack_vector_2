# Shutdown Accelerator
This layer will speed up existing timers that are counting down.

```
⇋ move 09
Entered node 09 
Shutdown amplifier accelerates existing shutdown timers. 
Countdown accelerated by 1 minute
```

If there are no running timers, this layer has no effect.

The effect takes place every time a hacker enters the node that contains this layer, even if the hacker had visited this node before.

## Hacking
Hacking this layer has no effect.

```
⇋ view 
Node service layers: 
0 OS 
1 Shutdown amplifier 

⇋ hack 1 
Hacking Shutdown amplifier reveals nothing new.
```
