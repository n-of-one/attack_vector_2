# Minesweeper ICE
This ICE is called Visphotak in game. Vishpotak is a rough rendition of the Hindi word for `explosion`.

![Picture of a minesweeper ICE puzzle](../../assets/players/ice/minesweeper.png)

As a hacker you are fixing corrupted memory of the ICE. Mines are represented as corrupt memory blocks. If you click on a corrupted memory block, you are blocked from making further moves. Your fellow hackers can still continue solving the puzzle.

## Minigame
Minesweeper ICE is the classic minesweeper game ([wikipedia](https://en.wikipedia.org/wiki/Minesweeper_\(video_game\))).

This version of minesweeper has guaranteed safe corners. You can always click these to get a good start.

## Resetting
You can reset the puzzle at any time by pressing the Reset button. You will need to keep pressing the button for 8 seconds. 

Resetting allows you to start over when all hackers have clicked mines.

![Picture of a minesweeper ICE puzzle with an exploded mine and a clear reset button](../../assets/players/ice/minesweeper-blocked.png)

## Multiplayer

Multiple hackers can work on the same puzzle at the same time.

Minesweeper has some inherent luck, sometimes there is not guaranteed safe move and you have to guess where a mine is. This can be mitigated by playing with multiple players, so if one player is blocked, the other players can continue solving the puzzle.

## Speedrunning
Minesweeper speedrunning is a popular sport, and the world record is under 30 seconds for the expert level. There are common patterns to learn that will greatly speed up the time needed to solve this puzzle.
