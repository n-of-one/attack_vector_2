# ICE overview
ICE blocks hackers:
- ICE layers block interaction with layers beneath the ICE layer.
- ICE blocks movement to nodes beyond the node that contains it.
- ICE blocks scanning of nodes beyond the node that contains it.

You need to `hack` it in order to progress through the site.

## ICE types
Hacking ICE consists of solving a puzzle minigame. You can do this alone or together with other hackers.

There are a number of different types of ICE each with a different puzzle minigame.
