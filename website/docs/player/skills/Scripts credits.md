# Scripts credits
This skill allows you to sell data for script credits, and use those to buy scripts. With this skill you can hack Credits source layers to steal their credits. See [here](Layers%20overview.md#credits-source).

Having this skill gives you access to the following pages:
- **Market**, here you buy scrips.
- **Credits**, here you transfer credits to other players and see your transaction history.

Optionally you can have an income defined with this skill. 

For more details, see: [Buying scripts](scripts/Buying%20scripts).
