# Preferences
By clicking on your name in the bottom menu, you can edit your see your account and edit your preferences.

## Account section
### Name
Set your hacker name. This is what other hackers will see. This is also used in the `/share` command.

### Type
Your account is a hacker type account. You cannot change this.

### Font size
This is the size of the font used in Attack Vector.

You can change the font-size at any time by pressing shift + page-up, or shift + page-down.

## Hacker section
### Character name
This is the name of your character. This is not visible in-game. This is only  used by game masters to more easily identify your account. 

### Icon
This is the image shown during a hack to identify you. Other hackers will see this image.

### Script credits
If you have the script credits skill, this will show how many credits you have.
