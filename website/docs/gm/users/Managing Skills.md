# Managing Skills
For a list of all skill, see [here](Skills.md).

## Assigning skills to players
You can assign skills for each player individually.

![](../../assets/gm/manage-skills.png)

Some specific skills can be added multiple times.

## Template user
You can define the default skills that all hackers will start out with. Do this by assigning those skills to the user called: 'template'. When a new hacker is created, it will receive the same skills as the template user.
