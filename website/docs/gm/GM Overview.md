# GM Overview
As a game master there are several things you can do in Attack Vector:

- Create sites to hack. See [here](sites/Introduction).
- Make ICE puzzles or switches available directly as standalone items. See [here](standalone/Standalone%20apps)
- Manage user accounts and skills. See [here](users/Managing%20users).
- Manage scripts. See: [here](scripts/Role%20of%20scripts).
