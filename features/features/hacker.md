# Hacker

Hackers are players.

---

# Home page

Allows hackers to start new runs or join existing runs

## Capabilities
- Search site to start new run
- View existing runs
	- join run
	- delete run

---

# Sites page

Only visible fi the hacker has the CREATE_SITE skill.

Same as the sites page for game master, see: `gm.md`
Can only see own sites.

 
---

# Scripts page

Only visible  if hacker has SCRIPT_RAM skill. 

Manage scripts of hacker

## Capabilities
- View RAM
	- Loaded scripts
	- Refreshing ram and next available block
	- Free RAM
- View list of owned scripts
	- Load
	- Unload
	- Offer for sharing
	- Delete
- Retrieve daily free scripts (script access for hacker)
- View balance, link to market


---

# Market page

Only visible  if hacker has SCRIPT_CREDITS skill. 

View and buy scripts from market.

## Capabilities
- View RAM
- View list of owned scripts
- View current credit balance
- View market offers (script access for hacker)
- buy script

---
# Credits page

Only visible  if hacker has SCRIPT_CREDITS skill. 

Banking interface for script credits.

## Capabilities
- View balance
- Retrieve script income on income days
- Transfer credits
- View list of transactions

---

# My account page

Hacker-configurable settings.

## Capabilities
- Edit font size
- edit character name (if enabled via config)
- edit username (if enabled via config)
- edit icon
- view script credits (if skill is owned)
- view skills (if enabled via config)
