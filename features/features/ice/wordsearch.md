# Word Search ICE

Word Search ICE presents a grid of letters with hidden words. The hacker must find and select the words in order.

## Capabilities
- Grid of letters with words hidden horizontally, vertically, or diagonally in any direction
- Words must be found in a specific order (word 1, then word 2, etc.)
- Click letters in sequence to select a word; matches forward or reversed spelling
- Correct selections are highlighted on the grid
- Current target word and progress (word X of Y) are displayed
- Grid size and word count scale with ICE strength
- Win condition: all words found in order
- Puzzle generation attempts up to 50 times for optimal layout
- WORD_SEARCH_NEXT_WORDS script effect can reveal upcoming words
- In testing mode, puzzle generation is seeded for reproducibility
- New word search with different words/grid generated on reset
